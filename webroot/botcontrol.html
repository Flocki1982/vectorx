Games


Roll a die

Vector rolls a 6 sided die

Voice command: Hey Vector, roll a die!
Vector will roll a 6-sided die and tell the outcome.



Try it!

Bingo!

Vector hosts a bingo game

Voice command: Hey Vector, bingo!
Vector will pull out numbers from 1 to 90 in random order. Rub Vector's back to get a new number. Click Vector's back button to quit.



Try it!

Pong

Play Pong against Vector using its light cube

Voice command: Hey Vector, let's play a classic!
Vector will start a Pong game. Place the light cube in front of Vector's screen and move it up or down to shift your paddle. The firt one to score 10 points wins.



Try it!

Rock Paper Scissors

Play Rock Paper Scissors against Vector

Voice command: Hey Vector, let's play a new game!
Vector will challenge you at "Rock, Paper, Scissors", trying to read your hand. Open hand = paper, fist = rock, two raised fingers = scissors.



Try it!

Language


How do you say...

Vector as an instant translator

Voice command: Hey Vector, how do you say <word> in <language> ?
Vector translates a given word in the given language.



Try it!

Let's talk in...

Change STT & TTS language

Voice command: Hey Vector, let's talk in <language> !
Talk to Vector and get replies in the given language.

Weather


How's the weather now

Check weather all around the world!

Voice command: Hey Vector, how's the weather [in <location>] ?
Vector checks the weather in the given location. Don't specify a location to use Vector's default location.



Try it!

Weather forecast

Weather forecast using Vector

Voice command: Hey Vector, how's the weather [in <location>] <time> ?
Vector checks the weather forecast in the given location at the given time. Don't specify a location to use Vector's default location.



Try it!
